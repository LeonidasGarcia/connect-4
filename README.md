# Connect 4

Juego clásico de Conecta 4 implementado con React, TypeScript y animaciones modernas.

## Características

- Tablero interactivo 7x6 con física de gravedad
- Sistema de puntuación con animaciones vibrantes
- Efecto dorado al ganar una ronda
- Diseño responsivo y moderno
- Animaciones suaves en títulos, puntuaciones y estado del turno
- Soporte para multijugador online (próximamente)

## Tech Stack

- React 19 + TypeScript
- Vite 8
- TailwindCSS v4
- Zustand (gestión de estado)
- Framer Motion (animaciones)

## Ejecutar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
